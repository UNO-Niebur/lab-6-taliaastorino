# Lab 6 – Collections and Text Analysis

## Purpose
Practice lists, dictionaries, and file processing.

## Files to Complete
- `DiceRoll.py`   — simulate rolling two dice; show counts and percentages
- `WordCount.py`  — count lines, words, characters in a file
- `WordIndex.py`  — build a word → line-number index (dictionary)

## Testing
- Run each program with multiple test cases (including empty and invalid file names)
- Verify counts and percentages are reasonable
